# docs/ — Projekt-Historie & Provenienz

Dieser Ordner enthält die **Entstehungsgeschichte und Herkunftsnachweise** des Projekts — also die
Dokumente, aus denen das Fundament und spätere Ausbaustufen hervorgegangen sind.

## Was hierher gehört

- **Briefs** — Auftrags- und Anforderungsbeschreibungen des CEO.
- **Bootstrap-Prompts** — die Prompts, mit denen das Fundament errichtet wurde.
- **Spätere Build-Prompts** — Prompts/Anweisungen, die das System weiter ausbauen.

Zweck: Jedes Tool (Claude Code, Codex, ChatGPT) und jeder Mensch soll nachvollziehen können, **warum** das
Repository so aussieht, wie es aussieht — als lückenloser Herkunftsnachweis.

> Hinweis: Die **verbindlichen Regeln** stehen weiterhin in `AGENTS.md` (Repo-Root), das **Protokoll** in
> `projekt_changelog.md`. Dieser Ordner ist **dokumentarisch** und ändert keine Regeln.

## Inhalt

|-------|-----------|
| `bootstrap-prompt.md` | Ursprünglicher Bootstrap-Prompt, aus dem das Fundament (Governance, Charta-System, 14 Agenten-Entwürfe) entstanden ist. |
